============================================================
ERROR HANDLING PATTERN
============================================================

Repository
   │
   ├─ Success ──► Right<Data>
   │
   └─ Failure ──► Left<Failure>
                     │
                     ├─ ServerFailure
                     ├─ NetworkFailure
                     ├─ CacheFailure
                     ├─ AuthFailure
                     └─ UnknownFailure

Provider returns Either<Failure, Data>
   │
   └─ UI handles both cases:
      ├─ failure ──► ErrorWidget
      └─ data ───► ContentWidget
